# Bài 2 - Tìm kiếm và đánh giá thông tin học thuật

Folder này chứa bài tập về kỹ năng tìm kiếm và đánh giá thông tin học thuật.

Nội dung chính:
- Xác định chủ đề tìm kiếm.
- Sử dụng toán tử tìm kiếm nâng cao.
- Chụp ảnh kết quả tìm kiếm.
- Đánh giá nguồn tin theo các tiêu chí cụ thể.
- Tổng hợp kết quả thành file PDF.

File quan trọng:
- `notes.md`: mô tả mục tiêu, quá trình, sản phẩm và kỹ năng học được.
- `san_pham/Bài_2.pdf`: sản phẩm cuối cùng của bài tập.
- `screenshots/`: nơi lưu ảnh minh chứng nếu được tách hoặc render từ PDF.

Khi tạo website, Codex cần dùng PDF làm nguồn chính và render 1-3 trang tiêu biểu để hiển thị trong phần dự án.
