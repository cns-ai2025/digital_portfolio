# Screenshots - Bài 2

Ảnh minh chứng của Bài 2 hiện đang nằm trong file PDF tại:

`../san_pham/Bài_2.pdf`

Codex không cần yêu cầu người dùng tách toàn bộ ảnh thủ công.

Khi xây dựng website, hãy đọc PDF và chọn 1-3 trang tiêu biểu nhất để render thành ảnh PNG/JPG đưa vào website.

Ưu tiên chọn trang có:
- câu truy vấn tìm kiếm nâng cao,
- ảnh chụp kết quả tìm kiếm,
- bảng đánh giá nguồn tin,
- phần phân tích độ tin cậy của nguồn thông tin.

Tránh chọn:
- trang bìa,
- trang chỉ toàn chữ chung chung,
- trang không có minh chứng trực quan.
